# Ich lerne Deutsch

একটা ব্যক্তিগত জার্মান-লার্নিং জার্নাল সাইট — GitHub Pages-এ ডিপ্লয় করার জন্য তৈরি। তুমি (publisher) সাইট থেকেই নতুন আর্টিকেল পোস্ট করতে পারবে, আর হাইলাইট করা শব্দগুলো বোল্ড + amber রঙে দেখাবে।

## ফাইল স্ট্রাকচার

- `index.html` — পাবলিক সাইট (আর্টিকেল লিস্ট + রিডিং ভিউ)
- `admin.html` — Verfasser-Bereich / পোস্ট করার পেজ
- `style.css` — ডিজাইন (ink/parchment/amber থিম)
- `main.js` — পাবলিক সাইটের লজিক
- `admin.js` — পোস্ট করার লজিক (GitHub API কল)
- `articles.json` — সব আর্টিকেলের ডেটা (একটা উদাহরণ আর্টিকেল দেওয়া আছে)

## ১. GitHub-এ ডিপ্লয় করা

1. GitHub-এ একটা নতুন repository বানাও (যেমন `ich-lerne-deutsch`), **Public** রাখো (GitHub Pages free-এ পাবলিক repo লাগে)।
2. এই ফাইলগুলো repo-র root-এ আপলোড/পুশ করো।
3. repo-র **Settings → Pages**-এ যাও → Source: **Deploy from a branch** → Branch: **main** এবং folder **/ (root)** সিলেক্ট করে Save করো।
4. ১-২ মিনিট পর সাইট লাইভ হবে: `https://<তোমার-ইউজারনেম>.github.io/<repo-নাম>/`

## ২. পোস্ট করার দুইটা পদ্ধতি

### A) এক-ক্লিকে অটোমেটিক পাবলিশ (Token দিয়ে) — সুপারিশ করা হয়
এতে "Veröffentlichen" বাটনে ক্লিক করলেই সরাসরি তোমার repo-তে `articles.json` আপডেট হয়ে যায়, কোনো ম্যানুয়াল কাজ লাগে না।

1. GitHub-এ যাও → প্রোফাইল ছবিতে ক্লিক → **Settings → Developer settings → Personal access tokens → Fine-grained tokens → Generate new token**।
2. **Repository access**: "Only select repositories" → তোমার `ich-lerne-deutsch` repo সিলেক্ট করো।
3. **Permissions → Repository permissions → Contents** → **Read and write** সিলেক্ট করো (আর কিছু লাগবে না)।
4. টোকেন জেনারেট করে কপি করো (এটা একবারই দেখাবে, সেভ করে রাখো)।
5. সাইটে `admin.html`-এ গিয়ে **⚙ GitHub-Einstellungen** খুলে Benutzername, Repository-Name, এবং এই Token পেস্ট করে "সেটিংস সেভ করো" চাপো।
6. এখন থেকে ফর্ম পূরণ করে "Veröffentlichen" চাপলে সরাসরি পোস্ট হয়ে যাবে।

**নিরাপত্তা নোট:** এই টোকেন শুধু তোমার ব্রাউজারের `localStorage`-এ সেভ থাকে — কোথাও আপলোড/শেয়ার হয় না। তবু কাউকে দিও না, এবং শুধু এই একটা repo-র জন্য Contents permission দিয়েই বানাও যাতে অন্য কোনো ক্ষতি না হয়। শেয়ার করা/পাবলিক কম্পিউটারে এটা সেভ করো না।

`admin.html` পেজটা টেকনিক্যালি যে কেউ URL জানলে খুলতে পারবে, কিন্তু আসল পাবলিশ করার ক্ষমতা শুধু তোমার Token-এর মাধ্যমেই আসে — Token ছাড়া অন্য কেউ ফর্ম পূরণ করলেও সেটা তোমার সাইটে পোস্ট হবে না, সর্বোচ্চ একটা ফাইল ডাউনলোড হবে যা তাদের কোনো কাজে লাগবে না।

### B) ম্যানুয়াল (Token ছাড়া)
Settings পূরণ না করলে "Veröffentlichen" বা "শুধু JSON ডাউনলোড করো" বাটন একটা আপডেটেড `articles.json` ফাইল ডাউনলোড করে দেবে — সেটা GitHub-এর ওয়েবসাইটে গিয়ে রিপোর পুরনো `articles.json`-এর জায়গায় আপলোড/কমিট করে দিলেই নতুন আর্টিকেল লাইভ হয়ে যাবে।

## ৩. নতুন আর্টিকেল লেখার নিয়ম

- **Inhalt** বক্সে প্যারাগ্রাফ আলাদা করার জন্য মাঝে একটা ফাঁকা লাইন (Enter দুইবার) দাও।
- **Hervorzuhebende Wörter** বক্সে যেসব শব্দ বোল্ড+amber রঙে হাইলাইট করতে চাও, কমা দিয়ে আলাদা করে লেখো — লেখার মধ্যে ওই শব্দ যেখানেই থাকবে, অটোমেটিক হাইলাইট হয়ে যাবে।
- পোস্ট করার আগে "Vorschau" বাটনে চেপে দেখে নিতে পারো হাইলাইট কেমন দেখাচ্ছে।

## ডিজাইন

থিমটা একটা পুরনো পাসপোর্ট/অ্যাটলাসের ফিল্ড-জার্নালের আইডিয়া থেকে — গাঢ় ইঙ্ক-টিল ব্যাকগ্রাউন্ড, brass/amber রঙের stamp ব্যাজ প্রতি আর্টিকেলের তারিখে, Fraunces (সেরিফ হেডলাইন) + IBM Plex Sans/Mono (বডি/মেটাডেটা)। কালার বা ফন্ট পরিবর্তন করতে চাইলে `style.css`-এর `:root` ভ্যারিয়েবলগুলো এডিট করো।
